## PhD student in social cognition

Hello! I'm a PhD student in Cognitive Science at the École Normale Supérieure-PSL, supervised by Hugo Mercier. I study how people reason about what others know or believe. My work combines behavioral experiments with Bayesian and computational models of social inference.

All my experiments are preregistered, and code and data for each project live in the corresponding repository (see pinned repos). PDFs are available at my [personal website](https://mariusmercier.github.io/)

Happy to help if you run into trouble reproducing an analysis!
